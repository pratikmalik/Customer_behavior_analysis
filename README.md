# :bar_chart: Customer Shopping Behavior Analysis

## Overview
Customer Shopping Behavior Analysis is a data analytics project that explores customer purchasing patterns, product preferences, and revenue trends using Python, SQL, and Power BI. The project focuses on transforming raw shopping transaction data into meaningful business insights through data cleaning, exploratory data analysis (EDA), SQL-based analysis, and interactive dashboard visualization.

## Objectives
- Analyze customer demographics and purchasing behavior.
- Identify top-performing products and categories.
- Evaluate the impact of discounts and subscriptions on sales.
- Understand spending patterns across different customer segments.
- Create an interactive dashboard for business decision-making.

## Technologies Used
- Python (Pandas, NumPy)
- SQL (MySQL/MariaDB)
- Power BI
- Matplotlib & Seaborn
- Jupyter Notebook

## Project Workflow
1. Data Collection and Import
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. SQL-Based Business Analysis
5. Power BI Dashboard Development
6. Business Insights Generation

## Key Analyses Performed
- Revenue Analysis by Gender
- Category-wise Sales Performance
- Age Group Revenue Contribution
- Subscription vs Non-Subscription Spending
- Product Rating Analysis
- Discount Usage Analysis
- Repeat Customer Analysis
- Top Products Identification

## Dashboard Features
- Total Revenue KPI
- Total Customers KPI
- Average Purchase Amount
- Average Review Rating
- Revenue by Category
- Revenue by Age Group
- Gender-wise Revenue Analysis
- Subscription Status Insights
- Interactive Filtering and Visualization

## Key Insights
- Identified high-revenue customer segments and product categories.
- Evaluated customer spending behavior based on subscriptions and discounts.
- Analyzed review ratings to determine top-performing products.
- Generated actionable insights to support data-driven business decisions.

## Project Outcome
Developed an end-to-end analytics solution that demonstrates data cleaning, SQL querying, business intelligence, and dashboard development skills. The project provides valuable insights into customer shopping behavior and supports strategic decision-making through interactive visualizations.
